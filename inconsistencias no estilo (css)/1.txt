Sobrescrevendo estilos: Em alguns arquivos HTML, como python.html e validar certificado.html, você sobrescreve as cores do texto usando um bloco de estilo (<style>) que define a cor #230D73 para vários elementos. Isso faz com que a cor padrão branca do style.css seja ignorada. Para manter o código mais limpo e organizado, seria mais eficiente centralizar todos os estilos no arquivo CSS principal.
